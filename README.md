# ITDCHAT
Чат для итд.com

# Как установить.

1. Скачать расширение Tampermonkey
2. перейти на [browser://extensions/] для яндекса и [chrome://extensions] для хрома, там включть Tampermonkey
3. нажать на сведения, и поставить "Разрешить пользовательские скрипты"
4. Скачать [ITDCHAT.txt](https://raw.githubusercontent.com/EpsilonRationes/ITDCHAT/refs/heads/main/ITDCHAT.txt) ctrl + s, или кнопка в браузере.
5. Открыть Tampermonkey перейти в утилиты, нажать экспорт из файла и выбрать ITDCHAT.txt

Если у вас андроид, можно скачать приолжение Tampermonkey, или firefox он поддерживает расширения.

# Важно!

Личка работает через комменатрии у вас не должно быть бана на написание комментариев.

Создание чата работает через создания поста у вас не должно быть бана на написание постов. Писать в существующие чаты и принимать запрос на создание чата от другого пользователя можно и без этого. 

Не удалайте комментарий ITD-cross-tunel:reg и не выключайте уведомления для ответов на комменатрии и посты. Иначе вы не сможете создать чат.

# Несколько чатов и чат с собой

С каждым пользователем можно создать сколько угодно чатов. Можно создать чат с самим собой. 

# Удаление сообщений и чата

После создания чата когда оба пользователя получили ключ шифрования, то есть прошла одна минута. Можно удалять любые сообщения из чата. 

Сам чат может быть удалён на стене [kilama](https://итд.com/kilama) создатель чата (тот кто нажал на кнопку создать чат) может снисти весь чат, удалив соответствующий пост.

# Как работает

Создание чата занимает **одну минуту**, для этого нужно перейти в профиль пользователя, там будет кнопка "создать чат", как только чат создаться придёт уведомление. После чат появится во вкладке уведомлений. У второго пользовтеля тоже должно быть установлено расширение, и он должен быть в онлайн.

Второй пользователь должен быть онлайн только для создания чата, писать сообщения в уже созданном чате можно в любое время.

# Где хранятся чаты

Чаты храняться в итд, а конкертно на стене [kilama](https://итд.com/kilama) в зашифрованном виде. 

# Шифрование

Используется алгоритм диффи-хеллмана, доступ к ключам имеют только два пользоватяля чата. Ключи хранятся локально в браузере, никуде не передаются, в случае потери восстановить невозможно. 

Хранятся данные в localstorage вашего браузера, обратите внимание, что на некоторых устройствах данные могут удалться если не использовать сайт более 7 дней.

# Куда писать

На вкладке уведомлений, таб "упоминания" заменится на "чаты". В табе чатов нужно нажать на нужный чат, там и писать сообщение. Изначально в чате есть несколько сообщений для установления ключа шифрования. 

# Сообщения от расширения

Сообщения от расширения отображаются точно так же как и обычные уведомления итд с поментой ответ на комментарий, они негде не хранятся и удаляются при перезагрузке страницы.
