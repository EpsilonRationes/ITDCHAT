# ITDCHAT
Чат для итд.com

# Как установить.

1. Скачать расширение Tampermonkey
2. Скачать [ITDCHAT.txt](https://raw.githubusercontent.com/EpsilonRationes/ITDCHAT/refs/heads/main/ITDCHAT.txt) ctrl + s, или кнопка в браузере.
3. Открыть Tampermonkey перейти в утилиты, нажать экспорт из файла и выбрать ITDCHAT.txt

# Как работает

Создание чата занимает одну минуту, для этого нужно перейти в профиль пользователя, там будет кнопка "создать чат", как только чат создаться придёт уведомление. После чат появится во вкладке уведомлений. У второго пользовтеля тоже должно быть установлено расширение, и он должен быть в онлайн.

# Где хранятся чаты

Чаты храняться в итд, а конкертно на стене (kilama)[https://итд.com/kilama] в зашифрованном виде.
